# Xacari - AI Workout Coach Project Status

## Project Overview

Xacari is an AI-powered mobile workout coach that uses computer vision (pose estimation) to provide real-time form correction and voice feedback during workouts.

**Tech Stack:**
- **Mobile**: React Native + Expo
- **Backend**: FastAPI (Python)
- **Database**: Supabase (PostgreSQL + Auth + Storage)
- **AI Model**: MoveNet Thunder (TensorFlow Lite)
- **Voice**: Parler TTS (Hugging Face)
- **Deployment**: Docker

**Hardware Constraints:**
- Development PC: 8GB RAM, GTX 4GB GPU, Windows + WSL2
- Mobile: iPhone (testing), Android Emulator (Expo Go)

## Getting Started

### 1. Backend Setup (Phase 1 - DONE)

```bash
cd backend

# Activate virtual environment
source xacari_env/bin/activate  # Linux/Mac
# xacari_env\Scripts\activate    # Windows

# Install dependencies (already done)
pip install -r requirements.txt

# Set up environment variables
cp .env.example .env
# Edit .env with your Supabase credentials

# Run the server
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# Access API at http://localhost:8000
# Access docs at http://localhost:8000/docs
```
